Privacy

What We Collect

Writer's Toolbox stores only the data that you save in it. We do not collect any personal data. We do not collect any analytics.

What We Save

Writer's Toolbox only saves the data that you enter. It is only saved on the device that you enter it on. No data is sent to any outside server or other database.

When We Change

Should there be an update of Writer's Toolbox tha changes any of the above, it will be communicated clearly in the release notes of the version in which it changes. What was changes and the purpose for which it was changed will be clearly conveyed. This privacy page will also be updated to reflect the current situation and the version of the application where the change occurred.

We are not in the business of secretly collecting or selling your data. Any projected changes to this policy will be for enhancing the application, for example data synchronization.
